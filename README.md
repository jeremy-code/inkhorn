# inkhorn

![GitHub Actions badge](https://github.com/jeremy-code/inkhorn/actions/workflows/ci.yml/badge.svg)

## Installation

```bash
git clone https://github.com/jeremy-code/inkhorn.git
cd inkhorn
yarn
```

## Usage

```bash
# Run the development server locally
yarn dev

# Lint and format code
yarn lint
yarn format

# Build
yarn build
```
